Why Study Information technology at RMIT Online?

What is my interest in IT?

My interest in IT is pretty broad. I view IT to be solutions for human problems.

With help from the lovely people in the Ubuntu Forums, I installed Ubuntu Linux on a laptop and learnt about using the command line and the principles of Linux and GNU . When I was confident enough, I built a NAS as a test bed for server based applications, and as a home for my digital life.

I had an interesting career in the NSW public service. Through the course of my work I became interested in systems analysis and process design, databases, and user experience.

My current IT setup looks like this:

  • A dual monitor desktop running Windows 10
  • A laptop running Linux Mint
  • A NAS running an Ubuntu LTS
  • A few Raspberry Pi SBC's

When did my interest in IT start?

I've had a strong interest in computing since my dad bought a Commodore C64 (Links to an external site.) in the mid-1980's. Amiga (Links to an external site.)'s 500, 600 and 1200 followed which I used for school homework, games, coding, music production, and creating bitmap graphics.
